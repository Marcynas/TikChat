# TikChat
Chat PWA

## Set goals:
- 📱 Cross-platform support
- 💬 Ability to chat anonymously. 
- 🖼 Send images.
- 🕵️ Private messages. 
- 🫂 Friend list.
- ❗ Profile editing.

From the graphic design side, the goal is to make the app's environment as modern and intuitive as possible.
